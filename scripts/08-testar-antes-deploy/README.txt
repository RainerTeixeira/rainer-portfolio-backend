📋 DESCRIÇÃO
═══════════════════════════════════════════════════════════════════════════

Testa sua aplicação localmente com DynamoDB Local para garantir que tudo 
funciona ANTES de fazer deploy na AWS. Simula o ambiente de produção 
localmente, economizando tempo e evitando custos com deploys com erros.

✅ Validação completa pré-deploy
✅ Simula ambiente AWS localmente
✅ Testa com DynamoDB Local
✅ Verifica endpoints críticos
✅ Relatório detalhado de testes


🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════

WINDOWS - Opção 1 (Mais Fácil):
   Duplo clique em: testar-antes-deploy.bat

WINDOWS - Opção 2 (PowerShell):
   .\testar-antes-deploy.ps1

LINUX/MAC/WSL:
   chmod +x testar-antes-deploy.sh
   ./testar-antes-deploy.sh
