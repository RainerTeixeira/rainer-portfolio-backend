🚀 PARA COMEÇAR AGORA (Opção Mais Rápida):

   1. Abrir PowerShell nesta pasta

   2. Iniciar MongoDB:
      docker-compose up -d mongodb

   3. Aguardar 10 segundos

   4. Iniciar a aplicação:
      npm run start:dev

   5. Acessar:
      • API: http://localhost:4000
      • Swagger/Docs: http://localhost:4000/api

🛠️ COMANDOS ÚTEIS:

   Ver status dos containers:
   → docker ps

   Ver logs:
   → docker-compose logs -f mongodb
   → docker-compose logs -f dynamodb-local

   Parar tudo:
   → docker-compose down

   Ver qual banco está ativo:
   → .\scripts\alternar-banco.ps1 status

   Gerenciar ambiente:
   → .\scripts\docker-ambiente-completo.ps1 start
   → .\scripts\docker-ambiente-completo.ps1 status
   → .\scripts\docker-ambiente-completo.ps1 stop
